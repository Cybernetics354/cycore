`NumExtension` Extension for `num` data type.

## Utility

### rupiah()
**-> String**<br />
`String trailing`
`String separator = "."`<br />
Convert num type to rupiah, example : 1000 -> Rp. 1.000, you can add trailing with `trailing` parameter and separator with `separator` parameter.<br />
*non-nullable*
