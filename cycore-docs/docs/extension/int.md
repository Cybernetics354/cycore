`IntExtension` Extension for `int` data type.

## Utility

### intToTimeLabel()
**-> String**<br />
`bool useStringLabel = true`<br />
Convert from int to something like `2 Jam 3 Menit`, you can change become `3:02` with parameter `useStringLabel` set to false.<br />
*non-nullable*
